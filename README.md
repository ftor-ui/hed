# hed
Simple hex editor    
ESC - normal mode  
s - HEX mode    
a - ASCII mode  
In normal mode:  
h, j, k, l - left, down, up, right  
q - quit  
w - save    
@[offset in hex] - jump to offset  
/[word] - search after cursor and jump to first match  
?[word] - search before cursor and jump to first match  
In other modes:  
Backspace - return to previous byte  
u - undo last change  
r - cancel undo :/
